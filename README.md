# Nydd

## Introduction
A dark mini-platform/RPG/adventure game using Pyxel engine.

![](/images/prev00.gif?raw=true "")

## Dependencies
- [Python](https://www.python.org/) 3.7 or higher.
- [Pyxel](https://github.com/kitao/pyxel) 1.8.7 or higher.

## Controls
- Move: Arrow keys
- Attack: Z
- Jump: X
- Start game/Pause: Enter
- Quit to main menu: Q
- Exit game: Escape
